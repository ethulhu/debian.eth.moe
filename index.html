Eth's Debian repo

how to install

  1. add this repo's GPG key to your APT keyring. 
    
$ curl https://debian.eth.moe/public-key | sudo apt-key --kering /etc/apt/trust.gpg.d/debian-eth-moe.gpg add -
  2. add this repo to your APT sources. 
    
$ cat /etc/apt/sources.list.d/debian-eth-moe.sources
Architectures: amd64
Components: main
Signed-By: /etc/apt/trusted.gpg.d/debian-eth-moe.gpg
Suites: buster
Types: deb
URIs: https://debian.eth.moe/debian/
